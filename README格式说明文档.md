## 简介 | Introduction

一两句话简要描述这个项目是做什么的。  
可以是中文或英文，或者双语。

> 例如：这是一个基于 Python 的快速文件整理工具，可以根据文件后缀自动分类。

## 特性 | Features

- ✨ **轻量级**：无需依赖，开箱即用
- 🚀 **高性能**：支持多线程处理
- 📦 **跨平台**：支持 Windows、macOS 和 Linux

## 快速开始 | Quick Start

### 环境要求 | Prerequisites

- Python 3.8 及以上
- Node.js 14+ (如果是前端项目)

### 安装 | Installation

```bash
# 克隆仓库
git clone https://github.com/yourname/yourproject.git

# 进入目录
cd yourproject

# 安装依赖
pip install -r requirements.txt
